# Software Construction Collaborative Project

## Overview

This repository contains a **collaborative project for the Software Construction course**.
The project is developed as a team effort where each member contributes to the design, development, and improvement of the software.

The purpose of this repository is to manage the project's source code, track changes, and enable effective collaboration among team members using GitHub.

## Collaboration Workflow

This project uses GitHub for version control and collaboration. Team members contribute by:

1. Cloning the repository
2. Creating a new branch for their feature or update
3. Committing changes with clear commit messages
4. Submitting a Pull Request for review
5. Merging approved changes into the main branch

## Notes

All team members are encouraged to follow good coding practices, write clear commit messages, and communicate changes effectively to maintain a smooth development workflow.
